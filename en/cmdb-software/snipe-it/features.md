### **Features**

Snipe-IT has an excellent feature list and some of them are:

*   Open Source
*   2F Authentication
*   SAML Support 
*   Secure
*   Cross-Platform 
*   Slack Integration 
*   LDAP Integration 
*   REST API
*   Import/Export 
*   Activity Log Management
