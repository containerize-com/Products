System Requirements
-------------------

- HTTP web server Ex. Nginx, Apache, IIS, Oracle HTTP
- PHP 5.5+ for versions upto 0.4.1 , PHP 7+ for versions =&gt; 0.5.1 and PHP 7.1+ for versions =&gt; 0.6.1
- Oracle 12c or MySQL(Server version: 5.6.17+) or MariaDB
