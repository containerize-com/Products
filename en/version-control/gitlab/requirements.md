### **System Requirements**

GitLab is Ruby and PostgreSQL based open source version control software. It requires following software.

- Ruby 2.6 or greater
- Go 1.13 or greater
- Git 2.24 or greater
- Node 12.0 or greater
- Redis version 5.0 or higher
- PostgreSQL 9.6 or greater
