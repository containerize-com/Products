Installation Instructions
-------------------------

### Installing on Ubuntu

Run below commands.

 ```

$ apt-get install subversion
$ apt-get install libapache2-svn

```

### Installing on Red Hat Linux

Run following commands.

 ```

$ yum install subversion
$ yum install mod_dav_svn

```
