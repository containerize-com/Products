System Requirements
-------------------

The following key dependency packages are required to install the Publify:

*   Ruby on Rails 5.2.x
*   CRuby (MRI) 2.4, 2.5, 2.6 or 2.7
*   Rails: 3.x+
*   A database engine, MySQL, PgSQL or SQLite3
*   A compatible JavaScript installation for asset compilation
*   ImageMagick
*   RubyMine editor
*   Git
