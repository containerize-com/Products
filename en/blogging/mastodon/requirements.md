### System Requirements
-------------------

The following key dependency packages are required to install Mastodon best microblogging site:

*   Ruby 2.5+
*   Rails 6.1.3
*   PostgreSQL 9.5+
*   Redis 4+
*   Node.js 12+
*   Email delivery service
*   RubyMine editor
*   Git
