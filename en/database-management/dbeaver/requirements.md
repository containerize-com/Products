### **System Requirements**
-------------------

DBeaver requires Java (JRE) 1.8+ to run.

- Windows and MacOS installers have built-in JRE component
- But you have to install JRE on Linux manually
 
 ```
sudo apt-get install openjdk-11-jdk
```
