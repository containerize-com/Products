### **System Requirements**
-------------------

In order to install OpenSourceBilling, below are the system requirements

- Ruby v2.7.1
- Rails v6.0.2.2
- Bundler for gem versions
- MySql database
- Server: Heroku OR AWS
- RubyGems: Installation as dependency
- Editor: RubyMine
- wkhtmltopdf
- Git
