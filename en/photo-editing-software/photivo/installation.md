Installation
------------

- Open terminal (Ctrl+Alt+T) and run command to add the PPA:
 
 ```
add-apt-repository ppa:dhor/myway
```

- Update the system repository
 
 ```
sudo apt update
```

- Run following commands in terminal to install Photivo:
 
 ```
apt-get install photivo -y
```
