Installation
------------

- Open terminal (Ctrl+Alt+T) and run command to add the PPA:
 
 ```
sudo add-apt-repository ppa:ubuntuhandbook1/darktable
```

- Run following commands in terminal to install Darktable:
 
 ```
sudo apt update
```

 ```
sudo apt install darktable
```
