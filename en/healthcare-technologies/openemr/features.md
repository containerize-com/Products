#### **Features**

OpenEMR offers the following key features:

*   Open Source
*   Cross-Platform
*   Multilingual
*   e-Prescribing
*   CMS Reporting
*   Lab Integration
*   Schedule Patients Appointments
*   Integrated Billing
*   Track Patients Records
*   Backup Facility
*   Data Encryption
*   Integration With Third-Party Apps
*   Automated Reminders
*   User-Friendly UI
