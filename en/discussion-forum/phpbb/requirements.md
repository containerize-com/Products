### System Requirements

phpBB is PHP based online community builder software that support various databases. It requires following software.

- PHP 7.1.3+
- Any of following Database 
  - MySQL 4.1.3+
  - MariaDB 5.1+
  - MS SQL Server 2000+
  - Oracle
  - PostgreSQL 8.3+
  - SQLite 3.6.15+
- Nigix / IIS
