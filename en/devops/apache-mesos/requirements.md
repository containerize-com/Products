### **System Requirements**

Following are system requirements to install the apache mesos container management platform on your linux machine:

*   GCC 4.8.1+ or Clang 3.5+
*   kernel >= 2.6.28
*   OpenJDK
*   wget, tar and git
*   Open firewall port 5050
*   Linux 64 Bit
