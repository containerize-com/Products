#### **Features**

OpenVidu offers the following key features:

- Open Source
- Encrypted
- WebRTC Compatible
- Support For Front End Technologies
- Support For Back End Technologies
- Integrated Chat
- Multiparty Video calls
- Session Configuration
- Self Hosted
- Video Recording
- Multipurpose
- Customizable
- Multiplatform
- Video/Audio Filters
- Screen Sharing
