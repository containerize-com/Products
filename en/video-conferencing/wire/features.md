#### **Features**

Wire offers the following key features:

- Open Source
- Video/Audio calls
- Self Hosted
- Internal/External Conversations
- Encryption
- Support For Guest Rooms
- Chat Options
- Extensible
- Flexible
- User Friendly UI
- WebRTC Compatible
- File Sharing
- Secure
- Support For Third Party Apps
- Robust Business Model
