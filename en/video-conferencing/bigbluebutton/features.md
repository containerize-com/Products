### **Features**

BigBlueButton offers the following key features:

- Open Source
- Session Recording
- File Sharing
- Screen Sharing
- WebRTC Compatible
- Support For Third Party Software
- HD Audio
- Secure
- Multi-user Whiteboard
- Real-time Chats
- Support Emojis
- Polling
- Robust
- Parent / Student Portal
- Extensible
