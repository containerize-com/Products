### Features

Following are the key features of Lets-chat

- BYOS (bring your own server)
- LDAP / Kerberos Authentication
- XMPP Multi-user chat (MUC)
- Multiple Rooms
- Desktop Notifications
- @Mentions
- Private / Password-protected Rooms
- Image embeds
- File uploads
- Hubot Friendly
- REST-like API
- Multilingual (i18n)
- Transcripts / Chat History
