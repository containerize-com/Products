### **Features**
--------

Some of the great features of rapid7 metasploit framework are listed below:

*   The GUI environment
*   Support for testing large networks
*   Smart payload generation and switching
*   Automate penetration Test
*   Test with success
*   Gather and reuse credentials
*   Cleaner exits
*   Open source
