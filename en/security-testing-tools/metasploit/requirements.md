### **System Requirements**
-------------------

The following key dependency packages are required to install metasploit offensive security development environment:

*   Ruby: 2.7.2
*   Redis: 3.0+
*   Rails: 5.0.1
*   PostgreSQL
*   RAM: 4 GB
*   Disk space:1 GB
*   Ubuntu 14.04 or later
*   Processor: 2 GHz+ 
*   Bundler
*   RubyMine editor
*   Git
