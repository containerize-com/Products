Features
--------

Some of the great features of BeEF penetration testing framework are listed below:

*   The extension API
*   Restful API
*   Keystroke logging
*   Browser proxying
*   Integration with Metasploit
*   Hooking through QR codes
*   Phonegap modules
*   Plugin detection
*   Intranet service exploitation
*   Custom browser exploitation commands
*   Open source
