System Requirements
-------------------

Requirements to setup your own cloud with ownCloud include:

*   PHP 7.2, 7.3, and 7.4
*   MySQL Database
*   A fresh installation of Ubuntu with SSH enabled
*   You are connected as the root user
*   Your ownCloud directory is located in /var/www/owncloud/
*   Git
