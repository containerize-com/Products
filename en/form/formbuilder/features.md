### **Features**

Following are the key features of FormBuilder

- Multi-Pages Forms
- Drag and Drop UI
- Multi-Language Support
- Easy To Extend
- Written In ES6
- Custom Plugins Support
- Support For Popular JavaScript Front End Frameworks
- i18n Integrated
- Flexible
- Ability To Load External JS/CSS Files
