### **Features**

Some of the key features of Celery are listed below:

*   Scheduling
*   Monitoring
*   High availability 
*   Horizontal scaling
*   Autoreloading 
*   Resource Leak Protection
*   Workflows
*   Time and Rate Limits
*   Autoscaling
*   User Components
*   Open Source
