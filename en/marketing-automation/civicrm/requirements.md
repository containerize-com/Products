### **System Requirements**

CiviCRM is PHP and MySQL / MariaDB based marketing automation and CRM software. It requires following software.

- PHP 7.3
- MySQL 5.7.5+ / MariaDB 10.0.2+
- Drupal 7+ / Wordpress 4.9+ / Joomla 3.x.x / Backdrop 1.0+
