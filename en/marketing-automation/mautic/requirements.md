System Requirements
-------------------

Mautic is PHP and MySQL / MariaDB based marketing lead generator and campaign builder. It requires following software.

- PHP 7.2.21+
- MySQL 5.5.3+ / MariaDB 10.1+
- Apache 2.4 + / Nginx 1.8 + / Microsoft IIS 7
