#### **Features**

li3 RAD Framework provides wide-range of features. Key features are listed below.

- Open Source
- Multilingual
- Robust
- PSR-4
- Built-in Unit Testing
- Extensible
- Secure
- Multiple DB Support
- Global Search
- In Memory Storage
- Comprehensive Documentation
- Component Based
- Third-Party Integration
- ODM/ORM Support
