### Features

Helpy has all the advance features that you can imagine from top helpdesk software. Main features include

- Multichannel ticketing
- Knowledgebase
- Mobile-friendly
- Community Support Forums
- Embed Widget
- Multi-lingual
- Themeable
- Sends HTML email
- Customizable
- GDPR Compliant
