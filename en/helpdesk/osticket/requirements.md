### **System Requirements**

osTicket is PHP and MySQL based help ticketing system, you must have the following software

- PHP 5.6+ (7.3 recommended)
- MySQL 5.0+ (5.5 recommended) +
- Apache 2.4 + / Nginx 1.8 + / Microsoft IIS 7
