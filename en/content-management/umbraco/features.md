Features
--------

Following are the key features of Umbraco CMS

- Open Source
- Easy Development
- Multilingual
- Powerful Plugins
- Support For Media Files
- Secure
- Pre-Built Templates
- Smart Search Feature
- Complete Documentation
- Alive Community
- Microsoft Word Support
- User-Friendly UI
- Activity Log View
