### **System Requirements**

-------------------

In order to install Hypercable open source bi tool, you’ll need to install following dependencies and prerequisites:

*   Rails 6.0
*   Ruby-3.0.0
*   Puma App Server
*   PostgreSQL
*   SublimeText or RubyMine Editor
*   Git Version Control
