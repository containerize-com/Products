### **Features**

Superpowers offers the following key features:

*   Flexible
*   Syntax Highlighting
*   Extensible
*   Support for 3D Models
*   Sound Effects, Fonts
*   Live Error Reporting
*   Cross-Platform
*   2D, 3D Games
*   Support For Shared project
