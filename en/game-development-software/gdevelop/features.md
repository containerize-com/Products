### **Features**

GDevelop offers the following key features:

*   Extensible
*   Cross-Platform
*   Support For Multiple Effects
*   Integrations
*   Multi-Platform Live Preview
*   Robust
*   3D Objects Supported
*   Support For AdMob
*   Support For Xbox, PlayStation
*   Piskel For Designing
*   Jfxr For Sound Effects
