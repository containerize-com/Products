### **Features**

Godot offers the following key features:

*   REST API
*   Self-Hosted
*   2D,3D Support
*   Node-Based Architecture
*   Animations
*   Cross-Platform
*   Export Options
*   Extensible
*   Visual Scripting
*   Drag & drop UI
