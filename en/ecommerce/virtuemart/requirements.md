### System Requirements

VirtueMart is PHP, Joomla and MySQL based open source shopping cart software. Current requirements for VirtueMart include:

- Web Server (Apache / Nginx)
- Joomla 3.0 or greater
- PHP 5.3 or greater
- MySQL 5.5 or greater
