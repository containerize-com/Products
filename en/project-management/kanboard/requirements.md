### **System Requirements**

Kanboard free project collaboration software is PHP based and support various databases. It requires following software.

*   PHP 7.2+
*   Compatible Databases
    *   Sqlite >= 3.7
    *   Mysql >= 5.6
    *   MariaDB >= 10
    *   Postgresql >= 9.4
*   Apache / Nginx / IIS / Caddy Server
