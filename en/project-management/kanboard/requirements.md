System Requirements
-------------------

Kanboard is PHP based and support various databases. It requires following software.

- PHP 7.2+
- Compatible Databases 
  - Sqlite &gt;= 3.7
   
  
  - Mysql &gt;= 5.6
   
  
  - MariaDB &gt;= 10
   
  
  - Postgresql &gt;= 9.4
- Apache / Nginx / IIS / Caddy Server
