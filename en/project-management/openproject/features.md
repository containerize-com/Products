Features
--------

OpenProject has all the advanced features that you can anticipate from top agile and classical project management software. Key features include

- Online Gantt charts
- Work Packages
- Calendar
- Roadmap
- Product Timeline
- Simple Task Management
- Comments and History
- Watcher
- Agile Boards
- Backlogs
- Task Board
- Tracking
- Filters
- Workflows
- Custom Fields
- Time Tracking
- Cost Reports
- Budgeting
- Project wiki
- Contents
- Embed tables
- History
- Open Source
